# 购物车分析 (Market Basket Analysis)

基于Apriori算法的关联规则挖掘分析报告。

## 📊 分析概述

- **分析方法**: Apriori关联规则挖掘
- **数据规模**: 8,615条原始记录
- **清洗后**: 5,625条有效数据（保留率65.29%）
- **有效订单**: 1,620个
- **商品种类**: 187种

## 🔧 分析步骤

1. **数据加载与探索** - 检查数据结构和质量
2. **数据清洗** - 缺失值、异常值、重复值处理
3. **购物篮构建** - 按订单分组提取商品列表
4. **数据编码** - One-Hot编码
5. **频繁项集挖掘** - Apriori算法
6. **关联规则生成** - 计算支持度、置信度、提升度
7. **可视化分析** - 生成专业图表
8. **报告撰写** - 完整PDF报告

## 📈 主要发现

- **频繁项集**: 187个（全部为1-项集）
- **关联规则**: 0条（未满足置信度阈值）
- **原因分析**: 订单规模小（平均2.64件），商品分散度高
